# qa-test-cases

Manual QA test cases and bug report samples (portfolio repo).

## What's inside

- **bug-reports/**
  - `bug-report-template.md` — reusable bug report template
  - `sample-bug-report-login.md` — sample bug report (login issue)
- **test-cases/**
  - `login-test-cases.md` — manual test cases for login
  - `signup-test-cases.md` — manual test cases for signup + validation
  - [password-reset-test-cases.md](test-cases/password-reset-test-cases.md) — manual test cases for password reset
 
## How to use this repo

1. Start with the folder README files for quick context.
2. Use the **template** to write new bug reports consistently.
3. Review the **sample** to see a complete, professional report.
4. Check the **test cases** for positive, negative, and edge-case coverage.

## Skills demonstrated

- Writing clear **steps to reproduce** and **expected vs actual** results  
- Understanding **severity vs priority**  
- Creating structured **manual test cases** (happy path + negative + validation)  
- Organizing QA artifacts in a clean, recruiter-friendly format
